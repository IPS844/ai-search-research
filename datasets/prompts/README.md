# AI Search Query Library

This directory contains structured user query datasets used during AI Search research.

The goal is to build a comprehensive library of real world search queries that can be tested across ChatGPT, Gemini, Claude, Perplexity, Google AI Overviews, and traditional search engines.

Unlike artificial prompts, these queries are written to reflect how real users naturally search, compare products, evaluate brands, seek recommendations, and make buying decisions.

## Industries

The current query library covers:

* SaaS
* B2B Software
* Marketing Agencies
* Education
* Study Abroad Consultants
* Healthcare
* Skincare and Beauty
* Insurance
* Real Estate
* Legal

## User Intent Categories

Queries are organized by user intent instead of industry.

Current intent groups include:

* Brand Discovery
* Product Discovery
* Brand Comparison
* Competitor Comparison
* Product Alternatives
* Best Products
* Best Tools
* Buying Intent
* Reviews
* Trust Validation
* Educational
* Troubleshooting
* Local Recommendations

This structure allows the same user intent to be tested across multiple industries while maintaining a consistent research methodology.
