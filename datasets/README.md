# Datasets

This directory contains structured datasets used during AI Search research.

The goal is to make every experiment repeatable, measurable, and comparable across different AI search platforms.

These datasets are continuously updated as new experiments are completed.

## Current Datasets

### prompts.csv

A standardized prompt library used to test AI Search behavior across multiple industries, search intents, and AI platforms.

The dataset is designed to ensure that every experiment follows a consistent testing methodology.

## Planned Datasets

### Citation Dataset

Records every citation observed during experiments.

### AI Visibility Dataset

Tracks which brands appear across different AI search platforms.

### Trust Signals Dataset

Documents the trust signals commonly associated with brands that receive AI citations.

### Entity Dataset

Maps entities, relationships, topical authority, and semantic relevance.

### Community Signals Dataset

Measures the influence of Reddit, Quora, forums, GitHub, reviews, and other community platforms.

### Search Comparison Dataset

Compares AI Search results with traditional Google search results.
